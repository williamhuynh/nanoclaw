# AiD Normalised Internal Cost Rates

**Status:** Approved by Will, April 2026
**Purpose:** For Anurag Verma to use for internal costing purposes

## Blended Daily Cost Rates by Rank

| Rank | Daily Cost Rate |
|------|----------------|
| Director | $1,300 |
| Senior Manager | $1,000 |
| Manager | $800 |
| Senior Consultant | $625 |
| Consultant | $550 |

## Methodology

- Based on team spreadsheet with 12 members
- Single blended rate per rank (no split between consultant and data science tracks)
- Used highest base daily cost rate per rank, rounded up
- Contractor rates (e.g. Cynthia Evans) adjusted to equivalent salary + loading basis
- Loading factor: **1.1945x** (super 12%, payroll tax 5.45%, workcover 2%)
- Working days: **230/year**
- Manager level interpolated from $150k base salary (no current team members at that level)
